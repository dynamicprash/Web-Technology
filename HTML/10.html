Demonstrating Absolute and Relative URLs

1. Absolute URL

An absolute URL specifies the full path, including the protocol (e.g., http:// or https://) and domain name:

Visit Wikipedia (Absolute URL)

2. Relative URL

A relative URL specifies the path relative to the current document or directory:

  • Go to Image LIst (Relative URL)
  • View an Image (Relative URL)
